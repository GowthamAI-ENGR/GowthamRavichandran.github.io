

## Developer Portfolio — Dark & Modern

A single-page developer portfolio with smooth scroll navigation and subtle animations.

### Sections

1. **Navigation** — Fixed top nav with links to each section, smooth scrolling
2. **Hero** — Full-screen intro with name, title (e.g. "Full Stack Developer"), a short tagline, and CTA buttons (View Projects / Contact Me)
3. **About** — Brief bio with a profile image placeholder, key highlights/stats (years of experience, projects completed, etc.)
4. **Skills** — Grid of tech skills with icons, organized by category (Frontend, Backend, Tools)
5. **Projects** — Card grid showcasing projects with image placeholders, tech stack tags, descriptions, and links (GitHub / Live Demo)
6. **Testimonials** — Carousel of client/colleague quotes with names and roles
7. **Contact** — Contact form (name, email, message) with social media links (GitHub, LinkedIn, Twitter)
8. **Footer** — Copyright and quick links

### Design
- Dark background with subtle gradients
- Accent color for highlights and CTAs
- Clean mono/sans-serif typography
- Subtle fade-in animations on scroll
- Fully responsive (mobile-first)
- Placeholder content throughout so it's easy to customize
